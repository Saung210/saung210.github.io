RAIOne: Smart Inventory System with Chatbot

This is my final year capstone project with my four teammates. My role was developing a chatbot. RAIOne is a web and mobile application developed to address the inefficiencies of traditional student administration and inventory management systems in the Robotics and AI department at KMITL. By integrating modern technologies like React.js, NestJS, GraphQL, and PostgreSQL, the application streamlines equipment borrowing, student management, and approval workflows. The system also features an AI-powered chatbot using Python, GraphQL, LINE Messaging API and Line Frontend Framework (LIFF) to enhance user interaction and provide real-time assistance. This solution aims to improve efficiency, reduce errors, and enable scalable management of both student data and lab resources. The research focuses on creating an integrated platform that addresses gaps in existing systems.
