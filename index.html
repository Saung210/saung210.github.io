Projects

RAIOne (Smart Inventory System with Chatbot)

RAIOne is a smart inventory management system built for the Robotics and AI Engineering Department to digitize and streamline lab equipment borrowing and returning. The system features a user-friendly web interface built with React.js and TypeScript, and a robust backend using NestJS, GraphQL, and PostgreSQL with AI-powered Chatbot.
I was responsible for designing and developing the AI-powered RAIOne Line chatbot using Python and Ollama. I integrated it with the LINE messaging platform to automate user queries, support real-time inventory updates, and handle secure authentication using JWT. The project enhances the overall user experience by offering instant assistance on availability, borrowing policies, and deadlines, reducing admin workload and improving communication efficiency.

Danish - English Translation Model

This project explores various neural machine translation (NMT) approaches for translating Danish text into English. Using different frameworks and models, I experimented with both Transformer and LSTM architectures to optimize translation accuracy and efficiency. While some approaches demonstrated high accuracy, others highlighted challenges in tuning and computational demands.
